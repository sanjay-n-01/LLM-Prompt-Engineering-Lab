Author's Analysis

1

The Default Is Always Motivational Slop

When given a vague prompt (Tier 1), the LLM defaulted to surface-level motivational writing phrases like "hard work pays off" and "success follows dedication". Any nuance about life, career, or competitive skill evaporated without explicit framing.

2

Structure Commands Compliance, Not Depth

Tier 2 and 3 prompts produced structurally correct essays but the arguments remained predictable. The LLM filled the format without genuinely wrestling with the paradox. Structure improved the shape, but not yet the substance.

3

Philosophical Framing Unlocks Real Thinking

Tier 4 was the turning point. Philosophical frameworks like Wu Wei or Goodhart's Law produced ideas that felt genuinely constructed. The LLM is limited by the quality of the frame you give it.

4

The LLM Argues Both Sides With Equal Confidence

Tier 5 revealed that the LLM will argue any position with equal polish. It is a mirror, not a thinker. The quality of output is a function of the input.

5

Style Is a Genuine Lever

Tier 6 proved style is not decoration—it is a primary variable. The assigned voice determines how powerfully the message lands.

Conclusion: Prompt Engineering Is Thinking, Not Typing

"The single biggest lesson from 150 essays on one topic: the LLM is only as smart as your prompt is precise. A lazy prompt produces lazy output."

The LLM did not get smarter across these 150 prompts—I did.

First place in prompt engineering, it turns out, is a byproduct too—of thinking clearly about what you actually want to say.
